# Нормы цеха 188

Оффлайн-приложение для учета норм времени.

## Как использовать
1. Откройте приложение в браузере
2. Введите число и нажмите "Добавить"
3. Данные сохраняются автоматически
4. При появлении интернета нажмите "Синхронизировать"

## Установка на телефон
1. Откройте ссылку в браузере
2. Меню → "На экран «Домой»"
3. Готово!
